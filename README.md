# React Boilerplate

[![Build Status](https://travis-ci.org/firstthumb/react-boilerplate.svg?branch=master)](https://travis-ci.org/firstthumb/react-boilerplate)

## Installation

Update `.env.development` file

```bash
yarn install
yarn start
```
